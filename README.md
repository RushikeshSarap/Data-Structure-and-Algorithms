# README

This repository contains code in four languages: **C**, **C++**, **Java**, and **Python**. Below are the general steps/rules/commands to compile and/or run each type of file on Windows (PowerShell/CMD). Adjust file names as needed.

---

## C (`.c`)

### Compile
```powershell
gcc -o output.exe source.c
```

### Run
```powershell
.\output.exe
```

---

## C++ (`.cpp`)

### Compile
```powershell
g++ -o output.exe source.cpp
```

### Run
```powershell
.\output.exe
```

---

## Java (`.java`)

### Compile
```powershell
javac Main.java
```

### Run
```powershell
java Main
```

*(Replace `Main` with the class that contains `public static void main(String[] args)`.)*

---

## Python (`.py`)

### Run
```powershell
python script.py
```

*(Or `python3 script.py` depending on your Python installation.)*
